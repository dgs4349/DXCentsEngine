

sounds can be inferred.
jump {
	file: "jump.wav"
	v: 0.5,
}

containers must be explicit

Container {
	jump {
		v: 0.5,
		items: {
			...
		}
	}
}

containers can also have a name,
	just check for items key

Container {
	name: "jump",
	items: [
		...
	]
}

schemas

Container {
	jump {
		vol: 0.5,
		items: [
			Schema: {
				f: "jumps/|0-4|.wav"
			},
		]
	}
}
